이정란:
김예린: 파이썬에 비해서 java는 훨씬 어려웠다. 학습 동아리를 하면서 java를 공부했으니 진도에도 수월하게 따라갈 수 있겠다고 생각해 신청한 java 프로그래밍이라는 과목은, 학습의 주 교재로 사용했던 ‘생활코딩’의 영상과는 난이도도, 그 목적도 달랐기에 수업을 따라가는 데에는 ‘생활코딩’때와는 다르게 벅찼다. 이미 다 아는 내용일 거라고 생각한 것이 문제였다. 결론부터 말하자면, 나는 java 프로그래밍 과목을 철회하고, 동아리원들과 함께 생활코딩에서 가르쳐주는 내용에 집중했다. 이번에 ‘비특코인’활동을 하며 배웠던 내용과, 앞으로 스스로 공부해나갈 내용을 합친다면, 다음 학기에는 해당 과목에서 꼭 좋은 성적을 거두고 싶다. IT계열에 진출하지 않더라도, 문제를 해결하기 위해 알고리즘을 짜고 그것을 토대로 코드를 작성하고, 오류를 차근차근 해결해나간 후 완성된 결과물을 눈으로 본 경험 말이다. 특히 예전에는 특정 상황에 부딪혔을 때 막무가내로 일을 해결하려고 했는데, 자바와 루비, 파이썬을 간단히 배워 본 지금은 조금 더 차근차근, 알고리즘과 같은 방식으로 차분히 생각한 후 더 정교하고 효율적인 해결을 할 수 있을 것 같다는 생각이 든다. 마지막으로, 진로와 꿈, 목표에 대한 감을 잡을 수 있게 됐다. 전공 관련 분야에 취직하되, 취미로 대전 지역의 도서관과 관련된 정보를 모아둔 사이트를 만들어 이용자들에게 제공할 수 있으면 좋겠다는 생각이 들었다. 그러기 위해서는 코딩공부를 더 열심히 해서 사이트를 운영하고, 관련 데이터를 자동적으로 수집하여 뿌려줄 수 있는 기능을 구현해야겠지만.
