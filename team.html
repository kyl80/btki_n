팀 소개

1)팀명


팀의 이름인 비특코인은 ‘비전공자들의 특별한 코딩인생’의 줄임말이다.
코딩과는 전혀 관련 없는 전공을 가진 인문대, 경상대, 사회과학대의 학생들이 모였다.
말 그래도, 비전공자들이지만 활동을 통해 ‘특별한 코딩 인생’을 꿈꾸며 이름을 정하게 되었다.
“Everybody in this country should learn how to program a computer. Because it teaches you how to think.”
위 스티브 잡스의 말처럼 소프트웨어는 어디에나 존재하고 4차 산업혁명 시대를 살아나갈 우리는 코딩의 필요성을 느끼고 있다.
프로그래밍 언어(Python과 Java)를 배우는 것을 시작으로 미래인재에 한 걸음 다가서려 한다.

지금은 4차 산업혁명 시대이다. 지금 취업 시장에서 코딩이 반드시 필요한 것은 아니지만, 코딩 능력은 앞으로 점점 중요한 역량이 될 것이라고 판단하기 때문에 우리는 코딩 학습 동아리를 시작하게 되었다. 우리는 이 학습동아리 운영을 통해 앞으로 프로그래밍 언어를 배우고 코딩을 학습하며 단순 코딩 능력만이 아닌 문제를 해결할 수 있는 컴퓨터적 사고 능력을 키우는 활동들을 하고자 했다.

3)참여동기

학습 동아리 프로그램 참여 동기

충남대학교 홈페이지에서 학습 동아리 지원사업을 발견해서 지원하게 되었다. 신예슬 학우가 동아리원들을 모았으며, ‘이왕 공부할 거 계획서를 세워서 체계적으로 공부하자’는 마음가짐으로 학습 동아리 프로그램에 참여하게 되었다.


팀원 별 참여 동기


강미규: 코딩을 요새는 초등학생들도 배운다기에, 시대의 변화에 뒤처진다거나 적응하지 못할까 하는 두려움에 코딩을 배워야겠다고 생각하게 되었다. 그래서 다양한 컴퓨터 언어들을 접해보고 싶었고, 비전공자들끼리 코딩을 해본다는 말에 스터디에 참여해보게 되었다.
신예슬: 기본적인 html과 python ruby java 등의 컴퓨터 언어를 익혀 향후에는 웹,모바일 어플리케이션을 만들어 보고싶고, 머신러닝과 빅데이터 분야에 대해 깊게 공부해보고싶었다.
